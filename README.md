EcoTechTracker is a web app that tracks daily device usage and estimates the carbon footprint generated by laptops and mobile phones.

##  How to run locally
1. Download or clone the repository  
2. Open `index.html` in any modern browser (Chrome recommended)  
3. No installation required

##  Tech Stack
- HTML
- CSS
- JavaScript
- LocalStorage (for data persistence)
