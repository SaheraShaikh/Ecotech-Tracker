# Ecotech-Tracker
EcoTech Tracker — Tracks device usage and estimates carbon footprint in real time.
EcoTechTracker is a web app that tracks daily device usage and estimates the carbon footprint generated by laptops and mobile phones.

## How to run locally

1. Install Visual Studio Code
2. Download or clone this repository
3. Open the folder in VS Code
4. Open `index.html` in a browser  
   (Right click → Open with Live Server OR double click the file)


##  Tech Stack
- HTML
- CSS
- JavaScript
- LocalStorage (for data persistence)
